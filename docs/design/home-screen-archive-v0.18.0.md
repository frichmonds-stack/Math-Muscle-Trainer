# Home Screen Archive - v0.18.0

This archive note preserves the pre-dashboard Home direction before the first `Today's Training` dashboard pass.

The exact working implementation is preserved in the published `v0.18.0` snapshot:

- `docs/v15/index.html`
- `docs/v15/styles.css`
- `docs/v15/js/app-init.js`

## Archived Shape

- Brand-led Home surface with large `Math Muscle Trainer` identity.
- Utility buttons for About and Settings.
- Three launcher actions: `Learn`, `Start Workout`, and `Progress`.
- Small progress/momentum panels for daily stats and recent weekly activity.
- Startup was Train-first in `js/app-init.js`, so this Home screen was retained but was no longer the initial app surface.

## Replacement Direction

The active root app now uses Home as a native-app-style `Today's Training` dashboard: app identity, one primary daily-training CTA, quick workout choices, learning handoff, mastery/focus prompt, and a static training snapshot strip.
